# Exercise 2: Factory Method Pattern

## Objective

Implement the Factory Method Design Pattern for creating different types of documents.

The Factory Method Pattern provides an interface for creating objects while allowing subclasses to decide which class to instantiate.

---

## Components

### Product

Document

### Concrete Products

WordDocument

PdfDocument

ExcelDocument

### Creator

### Concrete Creators

WordFactory

PdfFactory

ExcelFactory

Centralized object creation
